How To Read This

  • Early: Intermediate materials and low-chain gems before the big alloy web opens up.
  • Mid: Everything early plus advanced non-endgame alloys, gems, and tech/magic materials.
  • Late: Full endgame pool, including depth cores, void alloys, matrix steels, and singularity-grade materials.
  • Build order: craft the listed main part for the desired Silent Gear blueprint/template, then add the listed tip, rod, grip, binding, coating, lining, cord, fletching, or setting when that gear type supports it.
  • Upgrade order: grade the main part first, then the tip/coating/lining, then utility parts. After the Silent Gear item is strong, finish it with Apotheosis sockets and affixes.
  • Close calls: if a recommended material is painful to make, use the alternatives column. Those are the next best scored materials for that same stage and purpose.
  • Raw late-game king: Infinite Singularity Core for damage-heavy tools and weapons.
  • Mining speed monster: Mekasonic Overdrive Core for high-speed mining setups.
  • Armor anchor: Infinite Singularity Core for toughness and survivability.
  • Magic focus: Ruby-Iridium Arcane Alloy for spellblade or enchantment-heavy play.
  • Use the guide as a build planner; exact in-game output still depends on part grade, blueprint, coating count, starlight charging, and Apotheosis affixes/sockets.
