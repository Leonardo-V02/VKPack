# Open Source Boundary

This repo can safely open-source pack-authored material: KubeJS scripts, custom configs, docs, and first-party GrindingGear source.

It should not publish third-party jars unless the project license and distribution terms clearly allow it. Prefer Modrinth-resolved dependencies or official upstream download URLs.

First-party GrindingGear jars may be attached to GitHub Releases because they are ours, but they are intentionally not committed to git history.
