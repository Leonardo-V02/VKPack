# Security Policy

Do not commit private worlds, server IPs, authentication files, logs, crash reports, Modrinth databases, account caches, or generated web/password configs.

Report security issues privately to the pack maintainer before opening a public issue.

Known intentionally excluded files include:

- `servers.dat`
- `usercache.json`
- `usernamecache.json`
- `logs/`
- `crash-reports/`
- `saves/`
- `config/resourceful-config-web.json`
- `config/Veinminer/update/*.jar`
- voice chat username/player volume caches
