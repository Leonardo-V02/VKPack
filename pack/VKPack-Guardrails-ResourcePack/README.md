# VKPack Guardrails Resource Pack

Small pack-owned resource fixes for VKPack.

This resource pack mirrors selected `kubejs/assets` compatibility overrides so they can be published or tested independently from the full modpack. It is intentionally narrow: no gameplay changes, no third-party assets, and no broad visual restyle.

## Included fixes

- DiagonalWalls + Architect's Palette model fallbacks for the five wall models that logged `Unknown value 'tall' for property 'north'` during model baking.

The same assets are also shipped through VKPack's KubeJS overrides for normal pack installs.
